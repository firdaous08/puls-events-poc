# POC : Assistant Intelligent Puls-Events (RAG)

Ce projet est un Proof of Concept (POC) visant à développer un assistant intelligent capable de recommander des événements culturels en s'appuyant sur l'API Open Agenda, en utilisant une architecture RAG (Retrieval-Augmented Generation).

## Structure du projet
- `requirements.txt` : Liste des dépendances Python.
- `.env` (à créer) : Contient les variables d'environnement sensibles (clés API).
- `test_imports.py` : Script de validation de l'environnement de développement.
